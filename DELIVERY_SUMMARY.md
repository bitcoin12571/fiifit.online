## 🚀 HOW TO RUN

### Start Development Server
```bash
cd /c/fiiifit_nutrition
npm install
npm run dev
```

**Visit**: http://localhost:3000

### Build for Production
```bash
npm run build
npm run start
```

### Deploy to Vercel
```bash
npm install -g vercel
vercel
# Select project + deploy
```
