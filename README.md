# 🌸 FiiFit Online - Platform Digitală Premium

## 📌 Descriere

FiiFit Online este o platformă SaaS de transformare feminină bazată pe metodă științifică.

- 🎬 **Academia**: 24 lecții video + 1040+ antrenamente
- 📊 **Dashboard LMS**: Tracking progres
- 💪 **Comunitate**: Suport 24/7 Telegram
- 💳 **Checkout 1-click**: Stripe
- 📲 **Mobile First**: Optimizat telefon

---

## 🏗️ Tech Stack

**Frontend**: Next.js 14 + TypeScript + Tailwind CSS  
**Backend**: Node.js + Express (în progress)  
**Database**: PostgreSQL + Redis (în progress)  
**Payment**: Stripe API  
**Video**: Vimeo On Demand (DRM)  
**Hosting**: Vercel + AWS

---

## 🚀 Quick Start

```bash
npm install
npm run dev
```
